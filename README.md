## ¿Qué es DevSecOps?

DevSecOps es una metodología que integra la **seguridad (Security)** dentro del proceso de **desarrollo (Development)** y **operaciones (Operations)** de software. 

Su objetivo es incorporar prácticas y herramientas de seguridad durante todo el ciclo de vida del desarrollo de aplicaciones, desde la planificación y el desarrollo hasta las pruebas, despliegue y monitoreo. De esta manera se identifican y corrigen vulnerabilidades de forma temprana, mejorando la seguridad y calidad del software.
